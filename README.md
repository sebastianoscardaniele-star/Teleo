# Club de Reseñas

App Vite + React + Supabase para que un grupo de amigos publique reseñas de películas, libros, series, aplicaciones y restaurantes.

## Deploy en Vercel

- Framework Preset: Vite
- Build Command: npm run build
- Output Directory: dist

## Variables de entorno en Vercel

Crear estas variables cuando tengas Supabase configurado:

- VITE_SUPABASE_URL
- VITE_SUPABASE_ANON_KEY

La app funciona aunque todavía no configures Supabase, mostrando datos de ejemplo.
